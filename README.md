# 🧵 Sistema de Gestão de Pedidos Artesanais

Projeto em desenvolvimento com foco em organização de pedidos personalizados, controle de clientes, produtos e acompanhamento do processo de produção.

---

## 🚀 Objetivo

Desenvolver uma aplicação backend para gerenciar pedidos de forma estruturada, aplicando conceitos de Programação Orientada a Objetos e evoluindo gradualmente para uma API REST com Spring Boot.

---

## 📦 Funcionalidades implementadas

- Cadastro de cliente
- Cadastro de produtos
- Criação de pedidos
- Adição de itens ao pedido
- Cálculo automático de subtotal por item
- Cálculo do total do pedido
- Controle de status com enum
- Simulação do fluxo completo via console
